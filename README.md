# Opposition-process-simulation
Statistical analysis and simulation of a multi-stage competitive selection process using ranking data, score distributions, and probabilistic modelling.

# Project Overview
This project explores the dynamics of a competitive public selection process through:

- exploratory data analysis
- ranking analysis
- score distributions
- probabilistic simulation
- and candidate progression modelling

The objective is to study how future stages and uncertainty may affect final rankings and placement probabilities.

# Current Status

The project is currently under development.
Planned components include:

- exploratory data analysis (EDA)
- ranking evolution analysis
- Monte Carlo simulations
- probability estimation
- and an interactive Shiny dashboard for visualization.
